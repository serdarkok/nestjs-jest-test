## Description
It performs some tests with mock data in "sort.service.spec". It sorts by making a calculation with values such as price, stock, page views of certain products on an e-commerce site.

## Installation

```bash
$ nvm use
$ yarn
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:watch
```
